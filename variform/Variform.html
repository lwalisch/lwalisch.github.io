# Methods

# generateData()

Use the information from the dataMapping fields in FormElementData to create a formatted output of the content captured in the application form.

Syntax

async generateData(formElementData: FormElementData, isRoot: boolean = true): void
1

Parameters

  • formElementData: FormElementData
    The formElementData containing the data entered by the user

  • isRoot: boolean = true
    This is not meant to be changed, as it is used internally for recursive execution of this method
